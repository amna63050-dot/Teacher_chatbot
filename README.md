# 🎓 Teacher Chatbot – Father Occupation Assistant

## 📌 Project Overview
Teacher Chatbot is a Python-based chatbot developed to provide information about my father's profession as a Mathematics teacher. The chatbot answers user questions using keyword-matching techniques and supports both Urdu and English languages.

## 🚀 Features
- Urdu and English language support
- Interactive GUI using Tkinter
- Keyword-based question answering
- Text-to-Speech functionality
- Information about teaching profession
- User-friendly interface

## 🛠 Technologies Used
- Python
- Tkinter
- pyttsx3
- VS Code
